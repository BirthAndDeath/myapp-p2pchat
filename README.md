## 安装与运行

### 环境要求

- Node.js (>=18)
- npm
- Rust 工具链

### 开发环境搭建

```bash
# 克隆项目后进入目录
cd myapp

# 安装依赖
npm install

# 启动开发服务器
npm tauri dev
```

你也可以进入cli目录然后执行

```bash
cargo run
```

单独运行cli事例

### 开发构建

```bash
# 构建
npm tauri build
```
